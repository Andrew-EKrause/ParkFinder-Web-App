About Park Finder

Park Finder is a web application designed to allow users to view different national parks and places in the United States. The National Park Services (NPS) API is utilized in the framework of this website and grants users access to over 460 locations across the U.S.

One of the features of this website is planning places to visit. If a user wishes to visit a certain state, they can view all of the nationally recognized locations in the state and determine which areas to vist. The user can also create an account with the Park Finder site to add parks and places to a built-in list on their user profile.

The Park Finder website was created for a semester project in a Python programming class (CS 224) at the University of Wisconsin - La Crosse. We hope that you enjoy exploring the website, and we intend to incorporate additional features in the future.
